# إدارة صندوق عمارة الريان 4

## نظرة عامة على المشروع

هذا المستودع يحتوي على الواجهة الأمامية (Frontend) لتطبيق إدارة صندوق عمارة الريان 4. تم بناء التطبيق باستخدام HTML و CSS و JavaScript خام (Vanilla JS) ليكون تطبيقاً خفيفاً وسريعاً لإدارة البيانات والعمليات المتعلقة بالصندوق.

**اسم المشروع:** amarah-admin
**التقنيات المستخدمة:** HTML5, CSS3 (مع دعم الوضع الداكن/الفاتح), Vanilla JavaScript.

## الميزات الرئيسية

*   **واجهة مستخدم بسيطة:** تصميم نظيف ومرتب يركز على سهولة الاستخدام.
*   **الوضع الداكن والفاتح (Dark/Light Mode):** دعم مدمج للتبديل بين الوضعين لتجربة مشاهدة مريحة.
*   **شاشة تسجيل دخول:** واجهة مبدئية لتسجيل الدخول (تحتاج إلى ربط بالخلفية).
*   **عرض الإجماليات والمدخلات:** مكونات لعرض ملخصات البيانات وإدخالات السجل الفردية.

## كيفية التشغيل محلياً

بما أن المشروع هو تطبيق ويب ثابت (Static Web App)، فإن تشغيله بسيط للغاية:

1.  **استنساخ المستودع:**
    ```bash
    git clone https://github.com/AH1099/amarah-admin.git
    ```
2.  **فتح الملف:**
    انتقل إلى مجلد المشروع وافتح ملف `index.html` مباشرة في متصفح الويب الخاص بك.

## النشر (Deployment)

يتم نشر هذا المشروع تلقائياً باستخدام **GitHub Pages**. سيتم تحديث الموقع المنشور في كل مرة يتم فيها دمج تغييرات جديدة في الفرع الرئيسي.
